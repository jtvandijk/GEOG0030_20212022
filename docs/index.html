Module Introduction


Welcome

Welcome to Geocomputation, a course that introduces you to both the principles of spatial analysis and the use of programming for data analysis.

Over the next ten weeks, youâ€™ll learn about the theory, methods and tools of spatial analysis whilst implementing small research projects, first using Q-GIS, and then using the R programming language within the R-Studio software environment. You will learn how to find, manage and clean spatial, demographic and socio-economic data sets, and then analyse them using core spatial and statistical analysis techniques.
